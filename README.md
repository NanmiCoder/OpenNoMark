## 📖 项目简介

`OpenNoMark` 是一个专注于 **AI 生成图片水印去除与画质恢复** 的开源项目。项目基于自研模型训练，支持对豆包、Google (Gemini)、OpenAI (ChatGPT/DALL-E) 等主流 AI 绘画平台生成的图片进行水印定位、智能去除与细节修复。

### 🔍 核心特性
- 🌐 **多平台适配**：内置主流 AI 生成水印特征库，支持自动识别与针对性处理
- 🧠 **自研模型架构**：基于 `[填写你的架构，如：Diffusion / SwinIR / ControlNet]` 训练，在去除水印的同时最大限度保留原始构图与纹理
- ⚡ **轻量易用**：提供 `CLI` / `Web UI` / `Python API` 三种调用方式，支持批量处理
- 🔓 **完全开源**：训练数据管道、模型权重、推理脚本与评测指标全部公开，欢迎 PR 与二次开发
